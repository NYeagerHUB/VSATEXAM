📄 Hướng dẫn định dạng JSON & nhập đáp án ▼

File JSON cần có cấu trúc {"questions":[...]}. Mỗi câu hỏi như sau:

{
  "type": "mcq",
  "question": "Câu hỏi...",
  "options": ["A...","B...","C...","D..."],
  "answer": 0
}

{
  "type": "truefalse",
  "question": "Câu hỏi...",
  "statements": ["Mệnh đề 1","Mệnh đề 2","Mệnh đề 3","Mệnh đề 4"],
  "answers": ["D","S","D","S"]
}

{
  "type": "short",
  "question": "Câu hỏi...",
  "answer": "63"
}

{
  "type": "matching",
  "question": "Câu hỏi...",
  "left": ["Ý 1","Ý 2","Ý 3","Ý 4"],
  "right": ["A...","B...","C...","D...","E...","F..."],
  "answers": [0,1,2,3]
}

💡 Đáp án: MCQ – index (0=A), Đúng/Sai – "D"/"S", Trả lời ngắn – chuỗi so sánh, Ghép cột – index của cột phải. Có thể để trống và nhập sau qua nút ✏️ sửa.
